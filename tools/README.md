# tools/

Maintenance and workspace utility scripts.

## Layout

- `tools/maintenance/` Python helpers (imports, generators, packaging, audits)
- `tools/powershell/` PowerShell helpers
- `tools/keepfiles.md` keep-versioned policy for reproducibility

## Note on historical records

Existing historical manifests may reference old utility paths under `scripts/`.
Those records are preserved as historical evidence and are not rewritten retroactively.
